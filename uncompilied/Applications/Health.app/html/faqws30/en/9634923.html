How do I install the Wireless Scale?

Wireless Scale was designed so that it is very easy to install even if you don't have any technological literacy. A quick start guide with screenshots is provided in the box to help you get started. You'll basically need to connect the scale in Bluetooth to your iPhone, iPad or iPod touch, and to install and launch the free Withings app. The app then walk you step by step throught the few set up steps.
