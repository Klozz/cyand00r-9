Einstufung von Blutdruckwerten durch die Weltgesundheitsorganisation (WHO) :

SYS DIA
Optimaler Blutdruck < 120 < 80
Normaler Blutdruck < 130 < 85
Etwas Bluthochdruck 130-139 85-89
Leichte Hypertonie 140-159 90-99
Mittlere Hypertonie 160-179 100-109
Schwere Hypertonie > 180 > 110

Diese Einstufung basiert auf den Blutdruckwerten, die bei Personen in sitzendem Zustand in Ambulanzen von Krankenhäusern gemessen werden.

note : Es wird jedoch davon ausgegangen Definition der Hypotonie. Es wird jedoch davon ausgegangen, dass diejenigen, deren systolischer Druck unter 100 mmHg liegt, an Hypotonie leiden.


Neuere Forschungsergebnisse legen nahe, dass für zu Hause vorgenommene Messungen die folgenden Werte als Hinweis für zu hohen Blutdruck angesehen werden können.

Systolischem druck> 135 mmHg
Diastolischem druck> 85 mmHg

Diese kriterien gelten für die blutdruckmessungen zu hause.
